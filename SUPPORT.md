# Fairway Support

Thank you for using Fairway.

## What Fairway does

Fairway helps golf groups create rollups, customise course details, log Stableford scores, and review player and group leaderboards.

## Need help

If you need support with Fairway, email:

`podavies@msn.com`

Support requests can include:

- app issues or bugs
- questions about score entry or saved rounds
- help understanding rollup scoring screens
- privacy or data questions

## Common help topics

- setting up a new rollup
- editing course and tee details
- logging scores after a round
- understanding player and group leaderboards
- restoring the default course template

## Current platform

Fairway is currently being released through Apple TestFlight and the App Store.
